
# Create Project

## Scala

```bash
 mvn archetype:generate                               \
      -DarchetypeGroupId=org.apache.flink             \
      -DarchetypeArtifactId=flink-quickstart-scala    \
      -DarchetypeVersion=1.8.0
```

## Java

```bash
mvn archetype:generate                                \
      -DarchetypeGroupId=org.apache.flink             \
      -DarchetypeArtifactId=flink-quickstart-java     \
      -DarchetypeVersion=1.8.0
```
